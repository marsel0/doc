title: "PayIn. Получение информации по заявке"

### Описание
Метод используется для получения информации по заявке. В query запроса добавляется параметр ID, тело запроса остается пустым.


## Структура запроса

* Метод: `GET`
* Endpoint: `$BASE_URL/public/api/v1/shop/orders/{id}`


## Header Parameters (Заголовки запроса)

| Параметр      | Обязательное поле | Тип    | Значение              | Описание                    |
|---------------|-------------------|--------|-----------------------|-----------------------------|
| Authorization | Да                | string | Bearer **shopApiKey** | Токен авторизации магазина  |
| Content-Type  | Да                | string | application/json      | Формат данных запроса (JSON)|


## Path Parameters (передаются в URL)

| Параметр | Обязательное | Тип    | Описание  |
| ---- | ------------ | ------ | --------- |
| id   | Обязательно  | string | ID заявки |

### Пример запроса

```bash
curl --location --request GET '$BASE_URL/public/api/v1/shop/orders/545fca1d-5db0-421a-9d3b-50fc2121aecd' \
--header 'Authorization: ••••••'
```
-------------------
